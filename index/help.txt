## Tickers: Returns a list of the top tickers across discord and checks whether your ticker is valid.
```/tickers <action> <ticker>```
- action (**Required**): Desired action to use.
- top [action]: Return top 50 tickers today.
- check [action]: Returns whether ticker is valid and provides closest match.
- ticker (*Optional*): Only required for checking.

## Prediction: Takes a ticker and returns a 3mo prediction probability.
```/predict <ticker> <model>```
- ticker (**Required**): Valid Yahoo Finance ticker. (use /tickers to validate)
- model (*Optional*): Choose from Implied Volatility, Extrapolation, Aggregrate-Extrapolation, and Logical Analysis. ([more details](https://github.com/razoring/QuantumDiscordBot))

## Quote: Tracks ticker infomation live; use refresh to quickly update
```/quote <ticker>```
- ticker (**Required**): Valid Yahoo Finance ticker. (use /tickers to validate)

## Chart: Returns the live quote and candlestick chart of the tickers up to the duration.
```/chart <ticker> <duration>```
- ticker (**Required**): Valid Yahoo Finance ticker. (use /tickers to validate)
- duration (**Required**): Choose from the list.

## Alerts: Set reminders for when your ticker reaches a certain price or return the daily volatility.
```/alerts <action> <ticker/identifier> <price>```
- ticker (**Required**): Valid Yahoo Finance ticker. (use /tickers to validate)
